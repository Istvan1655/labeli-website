F.A.Q

Comment créer une nouvelle page sur le site ?

Pour créer une nouvelle page, il faut tout d'abord éditer le fichier "src/Server.hx" et créer dans le classe Server une fonction doName où Name est le lien de la page. Cette fonction prend un paramètre : dispatch de type Dispatch qui contient les données de la requête HTTP. 

public function doApi(dispatch : Dispatch)
{
	result = "Contenu de ma page";
}

Comment accèder aux données ?

Dans la fonction qui décrit votre page, vous avez accès à l'objet api (voir ci-dessous). A partir de cet objet vous pouvez appeler directement les fonctions de l'api et ainsi récupérer les données du site.

Où écrire le code HTML ?

Pour intégrer du HTML dans votre page, il faut créer un template de page. Pour cela, créez un fichier .html dans le dossier src/templates. Vous pouvez ensuite rédigez dans ce fichier le contenu de votre page. Par défaut, ce contenu est inclus dans un élément HTML du template global.html, il ne faut donc pas inclure les en-têtes HTML. Ensuite, pour intégrer le HTML à votre page il faut appeler l'appel suivant dans votre fonction de page. 

result = new templo.Loader("page.html").execute({});

Comment intégrer des données dans le code HTML ?

Dans la fonction de votre page, vous pouvez passer des données depuis l'api au template HTML comme ceci : 

var data = {awesomeData : api.GetAwesomeData()};
result = new templo.Loader("page.html").execute(data);
Pour afficher ces données dans le code HTML : 
<div class="awesome">::"::awesomeData::"::</div>

Compiler le site pour appliquer les changements

Pour compiler le site, il est nécessaire d'installer Haxe. Il suffit ensuite de d'exécuter la commande : "haxe build.hxml". Le résultat de la compilation est placé dans le dossier "www" et est prêt à être upload sur le site.
